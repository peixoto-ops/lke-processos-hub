# LKE Processos Hub

Sistema integrado de gestão de processos juridicos com API CNJ, Fabric patterns e Obsidian como Single Source of Truth.

## Visao Geral

Este projeto consolida a automacao de relatorios processuais (API Comunica DJE), padroes Fabric para analise juridica, e integracao com o vault Obsidian que serve como hub central de informacoes de clientes e processos ativos.
